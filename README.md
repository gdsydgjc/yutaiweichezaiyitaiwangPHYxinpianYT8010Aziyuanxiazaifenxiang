## 资源描述

YT8010A 是单对以太网物理层收发器（PHY），它实现 IEEE802.3bw 定义的 100BASE-T1 标准的以太网物理层。适合广泛的汽车应用，它使用标准 CMOS 工艺制造，实现物理层的功能在单对平衡双绞线上传输和接收数据。  可替代博通BCM89811车载以太网芯片。
